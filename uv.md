# Installing uv on macOS

## Quick Start

1. **Install via Homebrew:**

```bash
brew install uv
```

Or via the official installer:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. **Verify installation:**

```bash
uv --version
```
