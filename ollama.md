# Installing Ollama on macOS

## Quick Start

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

Or download the installer directly: [Ollama.dmg](https://ollama.com/download/Ollama.dmg)

## Verify Installation

```bash
ollama --version
```

## Pull a Model

```bash
ollama pull gemma3
```
