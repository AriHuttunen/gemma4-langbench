# Installing Unsloth on macOS

## Quick Start

1. **Install [uv](https://docs.astral.sh/uv/)** if not already available (e.g., via [Homebrew](https://brew.sh/): `brew install uv`)
2. **Install Unsloth** via uv:

```bash
uv pip install unsloth
```
