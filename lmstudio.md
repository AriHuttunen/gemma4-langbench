# Installing LM Studio

## Quick Start

1. **Download** the installer for your platform from [lmstudio.ai](https://lmstudio.ai/)
2. Run the installer or move the app to your Applications folder (macOS)
3. Launch LM Studio

For detailed installation steps, see the [official documentation](https://lmstudio.ai/docs).

## Load a Model

1. Open LM Studio and go to the **Discover** tab
2. Search for `gemma` and download a model variant
